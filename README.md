# iris-classifier-ag2

Projeto da disciplina AG2 (Engenharia de Computação e Software – INATEL) para treinar, avaliar e disponibilizar um modelo de classificação para o conjunto de dados Iris.  
O projeto realiza pré-processamento, treinamento do modelo, avaliação e permite prever a espécie de uma flor a partir de dados inseridos pelo usuário.
